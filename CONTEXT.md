# Fortudo Planning

Fortudo helps a room organize scheduled and unscheduled tasks while preserving a shared view of
the plan.

## Language

**Unscheduled list**: The room's global collection of tasks that do not have assigned times.

**Unscheduled sequence**: The room's shared, user-defined ordering of tasks in the Unscheduled
list. It does not assign times or change task priority.

**My order**: The display mode that presents the Unscheduled list according to the Unscheduled
sequence.

**Priority**: The display mode that presents the Unscheduled list using completion status,
priority, and estimated duration without changing the Unscheduled sequence.
